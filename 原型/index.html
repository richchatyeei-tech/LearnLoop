这题你错了第 1 题
一次函数 y = kx + b 的图像经过第一、二、四象限,则 k、b 的符号是(　)
你当时写的
k > 0, b > 0
🎯 先看看,为什么错了?

概念性 · 你会套公式,但卡在「看图判断增减」。图像从左往右是往下走的,说明 y 随 x 增大而减小 → k < 0;它和 y 轴交在上半轴 → b > 0。你把「经过第一象限」误当成了 k>0。

跟着改一遍一步步想
  1. 先看图像走势:从左上到右下 → 在下降
  2. 下降 → y 随 x 增大而减小 → k < 0
  3. 再看和 y 轴的交点:在上半轴 → b > 0
  4. 所以 k<0、b>0,选 B
