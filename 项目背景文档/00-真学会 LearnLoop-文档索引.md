## 产品一句话（全文档统一）

**真学会 LearnLoop** 是一款面向中小学作业与测验场景的 **作业后 AI 学习闭环产品**：基于同一份作业数据，为教师生成讲评与分层干预方案，为学生生成错因讲解、订正任务与针对性补练，并为家长提供可理解的协同摘要，让一次作业从「批完」走向「真学会」。

- **主 Slogan**：批完不算完，真学会才算。
- **副 Slogan**：一份作业，老师知道怎么讲，学生知道怎么会，家长知道怎么配合。

## 完整主链路（Demo P0）

```text
作业布置 → 学生作答提交 → 作业管理（提交进度）→ AI 批改归因 → 教师确认批改
→ 备课讲评 → 确认布置订正（交接点 A）→ 学生闯关 → 学情回收（交接点 B）→ 沉淀
```

**降级原则**：信息架构不降级（多班级、作业列表、阶段详情、流转条）；演示数据可降级（预置题库/OCR/部分学生作答）。

## HomeworkLoop 状态（用户语 ↔ 实现）

| 用户语（PRD §9.1 / UI） | 说明 |
| --- | --- |
| 待作答 | 已发布，学生作答中 |
| 待批改 | 作答齐，待/正在 AI 批改 |
| 待确认 | AI 已判分归因，待老师复核 |
| 待讲评 | 批改已确认，待备课讲评 |
| 订正回收中 | 已布置订正，回收掌握结果中 |
| 已闭环 | 真学会率达标或手动关闭 |
| 需二次干预 | 未达标，需再讲/再练 |
